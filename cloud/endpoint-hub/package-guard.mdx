Package Guard gives you visibility into package installation and usage activity across all your endpoints. It's powered by [pmg](/pmg/quickstart) — the open-source Package Manager Guard CLI — which intercepts package manager commands and syncs activity to SafeDep Cloud.

## Prerequisites

- Install PMG using the [quickstart guide](/pmg/quickstart)
- Have a SafeDep Cloud tenant ID and tenant domain (see [Cloud quickstart](/cloud/quickstart))

## Enable cloud sync

<Steps>
  <Step title="Run the PMG setup installer">
    ```console
    $ pmg setup install

    █▀█ █▀▄▀█ █▀▀  From SafeDep (github.com/safedep/pmg)
    █▀▀ █░▀░█ █▄█  version: v0.9.0 commit: c3a351

    ✓ PMG aliases installed successfully
      Installed to:  /home/user/.pmg.rc
      Config at:     /home/user/.config/safedep/pmg
      Restart your terminal or source your shell to use the new aliases
    ```
  </Step>

  <Step title="Enable cloud sync in the PMG config">
    Open the PMG config file at `~/.config/safedep/pmg/config.yml` (use `pmg setup info` if you need to confirm the path) and add the cloud sync configuration below, preserving any existing settings:
    ```yaml
    # ~/.config/safedep/pmg/config.yml
    cloud:
      enabled: true
    # other existing settings...
    ```
  </Step>

  <Step title="Login to SafeDep Cloud">
    Authenticate with your tenant details:
    ```bash
    pmg cloud login
    ```
    Enter your **tenant ID** and **tenant domain** when prompted.
  </Step>

  <Step title="Sync to SafeDep Cloud">
    ```bash
    pmg cloud sync
    ```
    Run this after login and whenever you want to push new local activity to SafeDep Cloud.
  </Step>
</Steps>

## Common sync patterns

- **After dependency updates** — run `pmg cloud sync` after major package installs or upgrades.
- **In CI/CD pipelines** — add `pmg cloud sync` as a post-step after PMG runs in your pipeline.
- **Scheduled** — use a cron job for automatic nightly syncs, for example: `0 0 * * * pmg cloud sync`.

## View package events in SafeDep Cloud

After a successful sync, open [app.safedep.io](https://app.safedep.io), select **Endpoints** in the sidebar, and choose your endpoint to review the **Package Events** timeline.
