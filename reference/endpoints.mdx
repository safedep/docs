These are the public SafeDep hosts you interact with directly — the console, the API planes, identity, the community API, and the hosted MCP server. Use it as the canonical reference for what each host does and how it authenticates. For how to authenticate against the API planes, see [Authentication](/cloud/authentication).

<Note>
This is not an exhaustive egress allowlist. Some integrations reach additional internal hosts — for example, enabling vet-action's comments proxy adds `ghcp-integrations.safedep.io`. Check the relevant integration's docs when configuring strict firewall rules.
</Note>

## Hosts

| Host | Role | Authentication |
|---|---|---|
| `app.safedep.io` | Web console — sign in, manage your tenant, and create API keys | Interactive login (OAuth/OIDC) |
| `api.safedep.io` | Data plane — package insights, scanning, and malware analysis (gRPC / ConnectRPC) | API key |
| `cloud.safedep.io` | Control plane — tenant, policy, and management operations | JWT |
| `auth.safedep.io` | Identity provider — OAuth2 / OIDC, issues and validates JWTs | OAuth2 / OIDC |
| `community-api.safedep.io` | Community API — public malware and package queries | None (keyless) |
| `mcp.safedep.io` | Hosted [Model Context Protocol](/apps/mcp/overview) server for AI coding tools | API key |

<Info>
API keys are created in the web console at [`app.safedep.io/settings/api-keys`](https://app.safedep.io/settings/api-keys). Your **tenant ID** is your tenant domain (e.g. `your-company.safedep.io`).
</Info>

## Notes

- **`app.safedep.io`** is the SafeDep Cloud console. It replaces the retired `platform.safedep.io` — that host is no longer in use, so update any old references.
- The **data plane** (`api.safedep.io`) and **control plane** (`cloud.safedep.io`) are a [gRPC API with a ConnectRPC facade](/api-reference/introduction), not REST. See the canonical schemas at [buf.build/safedep/api](https://buf.build/safedep/api).
- The **community API** (`community-api.safedep.io`) requires no authentication and is rate-limited under a fair-usage policy.
